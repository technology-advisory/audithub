# Auditoria365 v9 - Multicliente y multiauditoría

La demo incluye dos clientes y dos auditorías independientes:
- CLI-001-ISO27001-2026-01
- CLI-002-ENS-2026-01

Las evidencias físicas se organizan como:
`evidencias/<CLIENTE>/<AUDITORIA>/<MARCO>/<CONTROL>/<TIPO>/archivo.ext`

Los documentos subidos por el navegador conservan la misma ruta virtual en IndexedDB.

Todos los datos son de demostración.
